# projeto-logica
projeto aplicando conhecimento da materia

1. O Hub de Jogos (Menu Principal)
Crie um laço infinito (faca-enquanto) que exiba as opções: 1 - Jogo da Velha, 2 - Palavras-Cruzadas, 3 - Jogo da Forca e 0 - Sair.
Use a estrutura escolha/caso para direcionar o usuário.
Se digitar uma opção inválida, exiba um erro e repita o menu. O sistema só desliga se digitar 0.

status: feito

2. O Motor do Jogo da Velha (Opção 1)
O Jogo da velha é para 2 jogadores. Como o jogo tem no máximo 9 jogadas, crie um laço para que conte de 1 até 9.

Dentro do laço, use a matemática (o resto da divisão por 2: % 2) para alternar os turnos.

Se a jogada for ímpar, escreva: "Vez do Jogador X - Escolha uma posição". Se for par, escreva: "Vez do Jogador O - Escolha uma posição".

Leia a posição digitada e avise que foi registrada.

status: em desenvolvimento...

3. O Motor do Jogo da Forca (Opção 3)
O jogo da forca precisa reconhecer os erros dos jogadores.
Crie uma variável vidas valendo 6. Use um laço enquanto que rode enquanto as vidas forem maiores que 0.

Peça para o jogador digitar uma letra. Como ainda não temos as palavras, faça o sistema perguntar ao "juiz" se a letra está certa (S/N).

Se errar, diminua 1 vida e exiba quantas restam. Se a vida chegar a 0, exiba "Game Over".

status: pendente.

4. Palavras-Cruzadas (Opção 2)
Esta missão exigirá 3 cenários diferentes e conhecimentos avançados que veremos em breve. Por enquanto, se o usuário escolher a opção 2, apenas exiba: "Em desenvolvimento...".

status: pendente.
